# TCP/IP Stack in C++20

TCP/IP stack built from scratch, with data link, network, and application layers in operation.

Implementation steps:

1. Multinode topology emulation of routers and switches
2. Datalink layer implementation (L2 routing), including ARP
3. L2 switching (mac-based learning and forwarding)
4. Vlan-based mac learning and forwarding
5. Network layer implementation (L3 routing)
6. IP-Tunnelling
